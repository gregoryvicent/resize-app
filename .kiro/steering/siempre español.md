# Idioma del Proyecto

## Comunicación y Documentación
- Todas las respuestas en el chat deben estar en español
- Toda la documentación (README, comentarios de código, archivos .md) debe escribirse en español
- Los mensajes de commit deben estar en español
- Los comentarios explicativos en el código deben estar en español

## Código
- El código fuente debe escribirse en inglés (nombres de variables, funciones, clases, tipos, interfaces)
- Las constantes y enumeraciones deben nombrarse en inglés
- Los nombres de archivos y carpetas deben seguir las convenciones estándar en inglés

## Ejemplos
```typescript
// ✅ Correcto: comentario en español, código en inglés
// Función para obtener el usuario actual
function getCurrentUser(): User {
  return user;
}

// ❌ Incorrecto: código en español
function obtenerUsuarioActual(): Usuario {
  return usuario;
}
```
